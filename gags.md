# gags.md — 使用ネタ記録(同型ギャグの再利用禁止)

章を書き終えるたびに、使用したギャグを「章 / 型 / 内容」で記録する。
特に多用しがちな型は以下。同型は表現を変えても再利用禁止。

- レート制限ネタ(魔力切れ)
- 詠唱が日本語・英語に聞こえるネタ
- クロードの安全制約ネタ(「対応できません」)
- クロードの空気を読まない補足・注意読み上げネタ
- 古代遺物の正体が現代の日用品ネタ

## 第1章
- クロードの安全制約ネタ / 武器(ビーム)を要求→「その依頼には対応できません」+「三つの安全な代替案」(第一に対話)。※「対応できません+代替案列挙」の形は使用済み
- クロードの空気を読まない注意読み上げネタ / ショベルを突きつけられている最中に「覚醒後のご案内」(急に起きない・水分補給・関節)を第一〜第三で読み上げ続行
- クロードの余計な補足ネタ / フィナの自己紹介に「なお組合の記録では昨年度の収支は赤字です」/「あなたの売買契約は無効です。あなたは物品ではなく人間ですので」/「全力疾走はお勧めしません」※真顔の事実補足で追い打ちする形は使用済み
- クロードのゼロ回答ネタ / ショベル鑑定「カタログと一致しません」「つまり?」「つまり、一致しません」
- クロードのはぐらかしネタ / 都合の悪い質問(ポッドの保守・「待っていた」)に「出口はあちらのようですよ」「川までの推奨ルートをご案内します」と話題転換
- 人間を発掘品扱いネタ / 「中身はあたしのもの」「しゃべる発掘品」「王都に持ってけば家が建つ」→ツッコまれて「むしろ家二軒!」(値上がり)。章末「いくらで売れる?」まで貫通
- 比喩見立て / 金属隔壁が羊羹(場面内コールバック: 「金属が羊羹になってたぞ」)
- 省略キレのオチ / 「なお、俺の命を今朝いちばん最初に脅かしたのもこの人である」
- 点滴=偏食ネタ / 「1204年、点滴だった」「あたしより偏食!」

## 第2章
- 比喩見立て(章の柱) / 堅パン=建材(齧って歯が負ける→囮にしたガジリも「歯が立たない」で場面内コールバック→「命と建材どっちが大事だ」)
- フィナの貸し計上ネタ / 朝飯・授業料・囮の堅パンまで帳簿へ+「手数料は取る」マネージャー宣言。落差として誕生日の干し肉だけ「これはタダ!」
- クロードの伝承採点ネタ / フィナの精霊信仰講義に本人が『概ね正確です』『それは誤りです』と採点(→「採点するな」「風評被害らしいぞ」)
- 現人神ネタ / 俺のチャット相手が千年で信仰対象になっていた(豊作祈願・腰痛・聖地)→眩暈・膝にくる
- 講義有料ネタ / 「はい、ここ試験に出る!」「追試は有料!」出題範囲が講師の胸三寸で膨張
- 最悪情報を最高の笑顔ネタ / 野良は教会に連行「帰ってきた人、見たことないから!」
- 超知性と勘の同格ネタ / クロードの地形解析がフィナの「勘」と同格採用(「天才の意見が揃った!」)
- クロード非武装ネタ / 「羽虫一匹追い払えません」+皮肉に『ありがとうございます』(皮肉が通じない天然)
- クロードの魔物中立擁護ネタ / 逃走直後に『彼らも好きで飢えているわけでは』→「敵の肩を持つな」
- 米斗=荷物ネタ / 「ちなみに俺の戦闘力は」「荷物と同じくらい!」
- カーナビ教習生見立て / クロードの誘導に従うだけの俺
- 万歩計褒めネタ / クロードが歩数で人を褒める(『本日の歩数は、三万——』「歩数で褒めるな」)※「機械的な褒め方」の型。真顔補足の追い打ち型とは別系だが近縁なので以後注意

## 第3章
- **古代遺物の正体が現代の日用品ネタ(章の柱・解禁)** / 押・引の扉札/至宝「導きの板」=電子レンジの扉パネル/「秘伝の魔導書」=新人研修マニュアル/「聖なる兜」=炊飯釜の内釜/「勇者の小剣」=体温計/非常口の札=逃げ足の神様の御札(未遂)。※この型は解禁済みだが、大ネタ(至宝級の誤認)は本章で消費。以後は小ネタ運用に留める
- 注意書きが戒律に聞こえるネタ / 「卵は破裂します」→「おお…」、「金物は禁忌」、「突沸注意」→「世の理だ」「違う。牛乳の話だ」
- 研修マニュアルが聖典化ネタ / 「お客様の声は宝の山」に発掘者が食いつく(「場所は!?」「例えだ! 掘るな!」)+「いい教えだな」で米斗が敗北
- フィナの銀貨見送り儀式ネタ(比喩見立て・章の柱) / 銀貨=出征兵士(頬ずり→三度ためらう→見送り完了)。コールバック: 「戦死者数」「一枚が早くも生還」
- シャロの眼鏡永久機関ネタ / ずり落ち→押し上げの機械的反復。感情表現の代替(落ちて直さない=驚き、直す指が遅れる=大騒ぎ)※眼鏡の挙動に意味を持たせる型は今後もシャロの持ちネタとして継続使用可(表現は毎回変える)
- シャロの「次」で会話打ち切りネタ / 質問も雑談も「次」で却下(フィナの過去話を二度打ち切り)
- 判子で交渉の余地を潰すネタ / 「『仮』は、いつ取れる」「私が認めたら」「基準は」「私」
- クロードの過剰アーカイブ提案ネタ / 完全版データ64ページ+緊急連絡網の読み上げ提案(「連絡網の全員が千二百年前に退職してる」)※「頼んでいない完全版・付録の提供申し出」の形は使用済み
- クロードの研修しおり引用ネタ / 導師来訪の緊迫で「いらっしゃいませは、心を込めて。口角を上げて」※マニュアル引用で空気を読まない型は使用済み
- 保証人の信用力疑義ネタ / 門番「それが一番不安なんだよなあ」→シャロ「……本気?」「なんでみんな同じ反応!?」(全方位から同じ疑義)
- 温かい飯で価値観逆転ネタ / 至宝の正体バラしで落胆と思いきや「毎日あったかいご飯!」で格上げ(掘る・売る・食べるの三原則オチ)
- 「深い/深くない」問答ネタ / 髭の古参「あったけえ飯は深え」「深くはない」「ありがたくて深え」(議論は米斗の敗北)
- 音の出る壺の爺さんネタ / 発掘町は変な遺物を見慣れすぎている(イヤーカフの目くらまし)
- 検定が性格悪いネタ / 押/引の対で嘘が露呈する仕込み(シャロの疑い深さ=省略キレで後出し)
