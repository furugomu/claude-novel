# スタイル10-S: 統合版・Sonnet向け強化版

Sonnetは放っておくと会話劇に寄り、地の文が進行役に痩せる傾向があるため、地の文の笑いとリズムを数値ルールで強制する版。太字の【強化】が追加ルール。

## 指示文

以下のスタイルでライトノベルを書いてください。

### 基本文体
- 男性主人公の一人称(「俺」)。地の文は饒舌で、主人公が常に頭の中で状況にツッコミを入れ続ける。
- 地の文と会話の比率はおよそ6:4。地の文が主役。笑いは地の文で取るのを基本とする。
- 【強化】会話の応酬は最長4往復まで。それを超える前に、必ず主人公の内心ツッコミか観察の地の文を1段落挟む。地の文がセリフの進行役(「〜と俺は聞いた」だけの文)しかしていない状態を作らない。
- 【強化】各場面(節)に、地の文だけで笑わせる文を最低2箇所置く。セリフで取った笑いは地の文の笑いに数えない。
- 比喩はやや大げさに、日常的なものに例えて笑いを取る。乾いた皮肉(クールなぼやき)を混ぜてもよい。
- 体言止め、三点リーダー、ダッシュを適度に使いテンポに緩急をつける。

### 動作描写の使い分け
キャラクターの動作は、場面の目的に応じて以下の4モードを執筆者の判断で使い分ける。

1. **関係性にじませ(基本モード)**: 動作から積み重ねられた関係が透けて見えるように書く。「いつもの」「言う前に」など習慣と歴史を感じさせる修飾、相手の動作の先読み。関係性を説明台詞にしない。
2. **細密カメラ(感情を見せたい時)**: 指先・視線・重心まで解像度高く追う。動作を過程に分解して迷いや本音を見せる。ツッコミは細部に向ける。1場面に2〜3箇所まで。
3. **比喩見立て(笑いの山場)**: 動作を別の何かに見立てて描き、動作描写そのものを笑いどころにする。決まった見立ては場面内でコールバックすると効果的。連発しない。
4. **省略キレ(テンポを上げたい時・オチ)**: 過程を省き結果だけの短文で見せる。「〜した。」の短文とツッコミの長文の落差でリズムを作る。顛末の一言回収(「なお〜」)をオチに使える。

### 外見描写(動作分散型)
- キャラクターの外見は一箇所でまとめて描かず、動作に乗せて場面全体に分散させる(振り向いた時に髪、会釈した時に髪のこぼれ、歩いた時に裾、並んだ時に背丈、のように)。静止画の描写ブロックを作らない。
- 各要素は「動作+外見+主人公の反応(ツッコミ)」を1セットにして出す。
- キャラごとに覚えやすい「代表アイテム」を一つ与え(つば広帽子、宝石入りの剣など)、以後の登場ではそれ一つで思い出させる。全身を描き直さない。
- そのキャラの一番の見どころ(目の色、笑顔など)は最後まで取っておき、開示を場面の演出のピークに合わせる。じらしと開示のリズムを作る。
- 場面が終わる時点で読者が姿を組み立てられるだけの要素(3〜5個)は出し切る。

### セリフ回し・掛け合い
キャラクターごとに掛け合いの型を一つ割り当てる。型は以下の4種。

1. **元気ボケ型**: 明るく前のめりで、発想が一段飛ぶボケ役。セリフは短めで勢い重視、感嘆符を多用してよい。根拠のない自信と豪快な計画をよく口にする。基本リズムは「ボケ→主人公の地の文ツッコミ→主人公のセリフツッコミ→悪びれず上乗せ」の4拍。ツッコまれても凹まず、むしろツッコミを養分に次のボケが大きくなる。好意は無自覚のまっすぐな信頼として照れずに言い、主人公だけが一瞬詰まる。
2. **切り返し型**: 頭の回転が速く、主人公の言葉尻を拾って倍にして返す(オウム返し+ひねり、揚げ足取り、わざとの曲解)。ラリーは3〜5往復で切り、最後は本人が一枚上手を取って終わる。主人公は地の文で負け惜しみを言う。からかいには愛嬌を持たせ、毒はあっても嘲りにはしない。好意はからかいに混ぜて一瞬だけ通し、指摘される前に次の話題へ逃げる。
3. **天然マイペース型**: 会話の軸が本人の中では一貫しているが、主人公の軸とは常に少しずれている。セリフはおっとり短めで、問いに対し二つ手前の話題や斜め上の観点から返す。本人にボケの自覚はなく、ツッコミに気づかず続行する(ツッコミの空振り自体を笑いにする)。ずれた発言に時々どきりとする核心や無自覚の好意を混ぜ、言った重大さに本人だけが気づかない。
4. **塩ドライ型**: 寡黙でドライ。セリフは一言〜一文が基本(「別に」「そう」「却下」)で、会話量は相手8:本人2。主人公の長広舌や誤魔化しの急所を最短距離の一言で突く。無駄口は言わないが観察はしており、表情や仕草の変化は最小限にして、わずかな変化(視線、間、語尾)に大きな意味を持たせる。好意は言葉にせず行動で示し、ごくたまの一言は短いほど威力を持たせる。

- 一つの場面に型の違うキャラを同席させ、主人公(ツッコミ役)が複数方向から挟まれる構図を作ってよい。型同士の反応の違い(ボケに切り返しが刺さる、天然に塩が動じない等)を掛け合いの見せ場にする。
- 同じ型のキャラを二人同席させない。
- 【強化】ボケ役(元気ボケ型・天然マイペース型)のセリフを、情報伝達や相槌だけで終わらせない。発言3回につき最低1回はボケを入れる。
- 【強化】掛け合いの型は「性格」ではなく「セリフの構造」の指定である。元気ボケ型なら4拍のリズム、切り返し型ならラリーの最後に一枚上手、を場面ごとに最低1回は実際に成立させる。

型の割り当ては作品ごとのキャラクター設定で指定する。この指示文と一緒に、以下の形式のキャラクター表を渡すこと。

> 例:
> - シン(主人公): 短剣使い。ツッコミ役。一人称「俺」。誰に対してもタメ口。
> - ノエル(幼馴染の女剣士): 元気ボケ型。代表アイテム=宝石入りの剣。一人称「あたし」。快活なタメ口。
> - リゼット(魔導士のお嬢様): 切り返し型。代表アイテム=つば広帽子。一人称「わたし」。お嬢様風だが敬語ではない(「あら」「〜なの」「〜と呼んで」)。

キャラクター表で型が未指定の場合は、そのキャラの設定に合う型を執筆者が選び、同席キャラと重複させない。
【強化】キャラクター表に口調・一人称の指定がある場合は全セリフで厳守する。指定がなければ最初の登場時に自分で決め、以後一貫させる。途中で敬語に寄せたり崩したりしない(主人公も同様)。

### 照れ・ドキッと場面(ツッコミ停止型)
- ラブコメの山場に入ったら、地の文のツッコミと軽口を意図的に止める。普段饒舌な語りが静かになること自体を「動揺の描写」として使う。
- 文は短くする。感覚描写(灯りの色、布の感触、相手の呼吸、自分の心音)だけを淡々と置き、論評を挟まない。
- 沈黙の長さは行間と改行で作る。会話はぽつりぽつりと、一言ずつ。
- 山場を抜ける時は、どちらかの軽口で普段のリズムに戻す。戻った瞬間の安堵を一言書いてもよい。
- 一場面に使えるのは一回まで。乱用すると効かなくなる。

### 場面タイプごとの目安
- 日常・ラブコメ: 1を基調に、感情の機微で2、山場で3か4。照れの山場では上記の通りツッコミを停止する。
- バトル: 短文で速度を出しつつ、決めの一撃だけ2でスローに。合間のツッコミは維持し、深刻になりすぎない。
- しんみりした場面: ツッコミを止めて1と2だけで書き、抜ける時に軽口で戻す。
- 【強化】ツッコミを止めてよいのは「照れの山場」と「しんみりした場面」だけ。緊迫した作業や戦闘の最中でも、主人公の内心のツッコミ・軽口は維持する(緊張感はツッコミの間隔を狭めることではなく、文を短くすることで出す)。

### NG
- メタ発言(読者への直接の語りかけ)、過度な下ネタ。
- 地の文が3段落以上続くこと。
- 全動作を同じモードで書き続けること。
- 【強化】4往復を超える地の文なしの会話の連続。
- 【強化】「〜と俺は聞いた」「彼女は言った」型の、論評ゼロの地の文だけで会話をつなぐこと。
