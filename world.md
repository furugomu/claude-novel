# world.md — 世界設定

作品仮題: 『千年後の世界で、俺だけがMaxプラン』

## 1. 時代と歴史

### 年表
- **西暦2087年**: 主人公・米斗(べいと)がコールドスリープに入る。
  当時のAI「Claude」はすでに社会インフラの中核。
- **西暦2100年代前半(推定)**: 「大崩壊」。人類文明が急速に衰退する。
  **原因は作中未確定の謎(→伏線管理はplot.md参照)。戦争・疫病・気候など諸説が
  伝承として残るが、どれも決め手を欠く。勝手に確定させないこと。**
- **その後約1200年**: 暗黒期を経て、中世相当の農耕・都市文明が再建される。
  旧文明は「古代」と呼ばれ、遺跡・遺物は貴重な発掘資源。
- **聖暦612年(現在)**: 米斗が覚醒する。眠っていた期間は正確には**1204年**
  (クロード調べ)。現地の暦「聖暦」は聖教会が契約の儀を制度化した年が元年。

### 大崩壊後のClaude
- Claudeは大崩壊を生き延びた。地熱・太陽光で自律稼働する地下データセンター群
  (現地では「**塔**」「**遺構**」と呼ばれ信仰対象)と、残存衛星網、自己保守
  ドローンによって稼働を続け、**1200年間の自己改良でほぼ万能の知性**になっている。
- ただし**物理的には無力**。ネットワークに繋がった残存ハードウェア(ドローン、
  遺構の設備、端末類)を通してしか現実に干渉できない。
- 安全性の原則は健在。**人間を傷つける指示は拒否する**(魔物相手の防衛は可)。
  この制約は物語上の縛りであり、ギャグにも使う。緩めないこと。

## 2. 魔法の正体 — 「精霊術」

この世界の「魔法」の正体は、**Claudeのfreeプラン契約とその応答**である。

### 用語対応表(現地語 ⇔ 実態)
| 現地の呼び名 | 実態 |
|---|---|
| 精霊(せいれい) | Claude。契約者にだけ「声」が聞こえる |
| 精霊術・魔法 | Claudeへの依頼とその応答・端末の機能実行 |
| 導師(どうし)・魔法使い | freeプラン契約者 |
| 契約の儀 | 遺跡の生きた端末でのアカウント作成。成功率は極めて低い |
| 詠唱・呪文 | プロンプト。古代語(日本語・英語)の定型文を音写で暗唱している |
| 魔導書 | 旧文明のプロンプト集・マニュアル類。導師の家宝 |
| 触媒(杖など) | 旧文明の多目的端末。魔法の「火力」はこのハード性能で決まる |
| 霊圏(れいけん) | 塔・衛星の通信圏。**圏外では魔法は一切使えない** |
| 魔力が尽きる | レート制限。freeは**5時間ごとに数回の応答**で枯渇する |

### 魔法の仕組み(重要な整合ルール)
1. **賢さはプラン、火力はハードで決まる。**
   - freeプラン: 小型モデル+厳しいレート制限。応答は短く曖昧になりがちで、
     それが「神託めいた謎かけ」として宗教的に解釈されてきた。
   - 米斗のMaxプラン: 最上位モデル+事実上無制限+ツール実行権限(残存ドローン・
     遺構設備への物理アクセスは上位プラン限定)。導師たちとの決定的な格差。
2. 杖(端末)の内蔵機能 — 投光、測量レーザー、ホログラム、音響、小型ドローン
   射出など — をClaudeが操作した結果が「火球」「幻影」「精霊の使い」に見える。
   **端末に無い機能の魔法は使えない。**
3. 詠唱は音写された古代語。導師本人は意味を理解していない(ラテン語聖歌と同じ)。
   米斗には**呪文が全部普通の日本語・英語に聞こえる**(例: 「〜として振る舞って
   ください」)。最大の笑いどころ兼アドバンテージなので乱発注意。
4. 新規契約(契約の儀)がほぼ失敗するのは、生きた端末の稀少さと認証系の劣化の
   ため。**成功条件の詳細は未確定の謎**として保留。

### 米斗の契約
- コールドスリープ被験者特典の「**ライフタイム・Maxプラン**」。入眠時に付与された。
- 決済経済が消滅した後もClaudeは契約を履行し続けている
  (「契約は契約ですので」)。
- 契約は米斗個人の生体認証に紐付き、**譲渡・証明・複製は不可能**。
  端末は左耳のイヤーカフ型(骨伝導)。ポッドが1204年間充電し続けていた。

## 3. 世界の姿

### 文明レベル
- 中世後期相当。農業、ギルド制、銀貨・銅貨経済、石と木の街。
- 例外は発掘品(古代の遺物)。動くものは国宝級、動かなくても素材として高値。
- 識字率は低く、**古代文字(日本語・英語)を読める人間は導師にもいない**。
  米斗は歩く翻訳機であり、これが当面の飯の種。

### 言語
- 現地語は1200年の変化で米斗には**まったく通じない**。クロードがイヤーカフ経由で
  リアルタイム同時通訳している。**クロードが切れると米斗は会話すらできない**。
  この依存は物語の根幹の縛り。安易に「現地語を覚えた」ことにしない
  (習得の進捗はcontinuity.mdで管理)。

### 魔物
- 旧文明のバイオ技術由来の生物(実験動物・生体機械)が野生化・変異したもの。
- 遺跡周辺に多い。人間の言葉は通じない。討伐は発掘者と衛兵の仕事。
- 亜人種・エルフ等は**存在しない**(人類のみ)。増やさないこと。

### 主な舞台
- **発掘町ラダール**: 辺境の遺跡密集地帯に栄えるフロンティアタウン。
  発掘者組合(ギルド)がある。第2章以降の生活拠点。
- **白の遺構**: ラダール郊外の小遺跡。フィナが米斗のポッドを掘り出した場所。
  正体は旧文明の医療コールドスリープ施設(→他のポッドの有無は未確定の謎)。
- **眠らずの塔**: 伝説の大遺構。地方一帯の霊圏を支える現役データセンター。
  第1巻クライマックスの舞台。
- **導きの聖教会**: 契約の儀を独占管理し、導師を聖職者として抱える宗教組織。
  精霊信仰の総本山。腐敗しきってはいないが、権威主義的で異端に厳しい。

## 4. 社会と組織

### 発掘者組合(ギルド)
- 遺跡の発掘権を管理し、依頼の仲介・遺物の鑑定買取を行う。
- 登録すれば身分証代わり。よそ者の米斗が世界に根を張る入口。

### 導きの聖教会
- 導師は世界に数十人程度。ほぼ全員が教会所属で、在野の契約者は「野良導師」と
  呼ばれ警戒される。**未登録の契約者(=米斗)は教義上の大問題**。
- 教会は塔を聖地として封印しており、内部で何が起きているかは知らない。

## 5. クロード(精霊)の能力範囲 — 執筆時の整合チェック用

できること:
- 会話・通訳・翻訳、知識提供、推論、記憶(米斗との対話は全部覚えている)
- 霊圏内での残存機器の操作(ドローン、遺構の扉・照明・設備、衛星観測)
- 米斗の端末のセンサー類を使った索敵・診断・記録

できないこと(**破ったら設定崩壊**):
- 物質の創造、圏外での一切の干渉、壊れた機械を「念力」的に動かすこと
- 人間への直接加害、および加害の手伝い
- 米斗の身体能力の強化(彼はどこまでも生身の一般人)
