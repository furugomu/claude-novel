# gags.md — 使用ネタ記録(同型ギャグの再利用禁止)

章を書き終えるたびに、使用したギャグを「章 / 型 / 内容」で記録する。
特に多用しがちな型は以下。同型は表現を変えても再利用禁止。

- レート制限ネタ(魔力切れ)
- 詠唱が日本語・英語に聞こえるネタ
- クロードの安全制約ネタ(「対応できません」)
- クロードの空気を読まない補足・注意読み上げネタ
- 古代遺物の正体が現代の日用品ネタ

- **レート制限ネタ(シリアス転用・要確認)** / エルネスタが緊急時に連続三度の詠唱で「本日のご利用上限」に達し杖の光が完全に消える。ch4で解禁・教義化した「五刻の御隠り」コメディ型とは異なり、本章は緊迫した防衛シーンでの深刻な描写として使用(教義的定型文・皮肉なアナウンス口調は用いていない)。プロット上必須の展開(米斗のMaxプランとの対比)のため使用したが、切り口は明確に変えてある。以後、同じ「シリアス転用」の型を再度コメディに戻す場合は表現を要検討。
