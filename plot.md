# plot.md — 第1巻プロット

仮題: 『千年後の世界で、俺だけがMaxプラン』
全12章構成(1章 = 6,000〜9,000字、1セッション1章)。

## テーマ
- **「言葉こそ魔法」**: この世界の魔法の正体はプロンプト。的確に頼み、的確に
  聞く力が米斗の強さ。クライマックスの解決も腕力ではなく「言葉」で行う。
- **依存と自立**: クロードなしでは会話もできない米斗が、「クロードの操縦者」
  から「クロードの相棒」になっていく話。
- **喪失と再接続**: 全部を失った少年が、新しい世界に根を張り直す話。
  笑い9割、この芯が1割。

## トーン運用
- 基調はコメディ(style.md準拠)。しんみりは各章1箇所まで、引きずらない。
- クロードの安全制約・レート制限・霊圏は「縛りとして機能するギャグ」。
  ご都合で緩めない(world.md整合ルール厳守)。

## 章別プロット

### 第1章「1204年ぶりの朝」
- **目的**: 覚醒、フィナとの出会い、世界観の第一提示。
- **内容**: 白の遺構。フィナが「開かずの棺」をショベルでこじ開けると中から
  少年が。米斗視点で覚醒→言葉が通じないパニック→イヤーカフ経由でクロードが
  起動し同時通訳(「おはようございます。1204年ぶりですね」)。フィナは米斗を
  「しゃべる発掘品」として値踏みする。遺構を出ると、崩れた高速道路に苔と森
  ——文明が終わった風景。
- **伏線(新設)**: F1 ポッドだけが真新しく保守されていた/F2 フィナのショベル
  の異常な切れ味/F3 クロードの「お待ちしていました」という一言。
- **引き**: 「ここ、どこだ?」「あたしの縄張り! で、あんた、いくらで売れる?」

### 第2章「文明の値段」
- **目的**: 世界のルール学習、フィナとの関係の土台作り、初のしんみり。
- **内容**: 野営しつつラダールへ。フィナが世界の常識を教える(魔物、導師、
  精霊信仰)。米斗は「精霊=Claude」に気づいて眩暈。夜、家族の話題で
  ツッコミ停止のしんみり(1204年の意味が身体に落ちる)。フィナは詮索せず
  「じゃ、今日があんたの誕生日だ」と流す。道中で小型の魔物に遭遇、クロードの
  索敵+地形誘導で戦わずに撒く(クロードは戦闘不能の提示)。
- **伏線(新設)**: F4 フィナの両親の失踪(さらっと)。
- **引き**: 街道の先に発掘町ラダールの灯。

### 第3章「国宝は電子レンジの取扱説明書」
- **目的**: ラダール・ギルド導入、シャロ登場、飯の種の確立。
- **内容**: ギルド登録(シャロの塩対応)。無一文の米斗、鑑定依頼の山に
  「古代文字が読める」ことを披露——ギルドの至宝の石板の正体は家電の取説。
  秘伝の魔導書は社員研修マニュアル。シャロだけが静かに米斗の価値に気づく。
  鑑定人として仮採用され、当面の宿と飯を確保。
- **伏線(新設)**: F5 シャロの「元発掘者」の過去(匂わせのみ)。
- **引き**: ギルドの扉が開き、白銀の長杖の導師様がご来訪。

### 第4章「導師様は本日の魔力を使い果たしました」
- **目的**: エルネスタ登場、魔法=freeプランの種明かしをギャグで固める。
- **内容**: エルネスタ、遺物の真贋鑑定で米斗と衝突。彼女の「詠唱」が米斗には
  日本語に聞こえる(「あなたは優秀な鑑定士として振る舞ってください」)。
  切り返しラリーの応酬の末、彼女の精霊が魔力切れ(レート制限)でダウン、
  米斗がしれっと正解を出して面目を潰す。エルネスタ、米斗を「未登録の契約者
  では」と疑い始める。
- **伏線(新設)**: F6 教会が未登録契約者を追う理由(過去の「野良導師の実害」を
  一言だけ)。
- **引き**: エルネスタ「あなた、精霊の声が聞こえていますわね?」

### 第5章「相棒の値段」
- **目的**: フィナとの初仕事、ショベル伏線の育成、関係の深化。
- **内容**: フィナが持ち込んだ小遺構の発掘依頼に鑑定人として同行。クロードの
  図面解析×フィナの勘の合わせ技で当たりを引く。崩落事故のピンチをフィナの
  ショベルが異常な性能で切り抜ける(F2育成)。分け前を巡る漫才と、フィナの
  「両親が消えた遺構をいつか掘る」という目標の開示(F4育成)。
- **引き**: 発掘品の中に、白の遺構と同じ紋章。

### 第6章「契約の儀」
- **目的**: 契約の儀の見学=魔法システムの深掘り、エルネスタの過去。
- **内容**: 教会がラダールで契約の儀を執行、志願者は全員失敗。米斗はクロードと
  「なぜ新規契約が通らないのか」を検討(結論は保留=謎4維持)。エルネスタが
  千に一つを引いた過去と没落貴族の事情が明かされ、ただの嫌味な女から
  「努力の人」に印象が反転する。米斗、うっかり彼女の詠唱の「発音」を直して
  精霊の応答率を改善させてしまう。
- **伏線(回収)**: 第4章の疑いが確信寄りに。**(新設)**: F7 儀式端末に残る
  旧文明のログ表示(米斗だけが読めた断片)。
- **引き**: その一部始終を、査問官ギデオンが見ていた。

### 第7章「ポニーテールと琥珀」
- **目的**: ラブコメの山場(前半)。フィナ回。
- **内容**: 収穫祭。フィナに引き回される米斗。ゴーグルを外して着飾ったフィナの
  琥珀色の目が初開示(見どころ解禁)。ツッコミ停止型を1回使用。無自覚の直球
  (「ベイトがいると、ひとりで掘ってた頃より百倍楽しい」)で米斗が詰まり、
  軽口で日常に戻る。終盤、祭りの夜空に流星——ではなく軌道上の何かが減衰落下
  していくのをクロードが観測する。
- **伏線(新設)**: F8 衛星網の劣化=霊圏はいつまで持つのか。
- **引き**: クロード「ベイト。少し、悪い報せがあります」

### 第8章「暴走」
- **目的**: 中盤クライマックス。米斗の力が公になる。
- **内容**: 白の遺構の旧式警備システムが衛星の異常信号(F8)で誤作動、魔物の
  群れを町側へ追い立てる。避難誘導でエルネスタのfreeプランはすぐ魔力切れ。
  米斗が公衆の面前でMaxプランを全開——残存ドローン操作、遺構設備の遠隔制御、
  リアルタイム索敵——で無血の防衛に成功する。ただし「人を傷つけない」制約と
  火力ゼロの縛りの中で頭を絞る展開にする(腕力勝利禁止)。町は救われ、米斗は
  「大賢者様」扱いに。
- **伏線(回収)**: F8起点の事件。**(新設)**: F9 警備システムが守ろうとして
  いた区画(白の遺構の深部)。
- **引き**: 教会の紋章入りの馬車が町に入る。査問官の召喚状。

### 第9章「査問」
- **目的**: 教会との対決(言葉の戦い)。エルネスタの立場の転換。
- **内容**: ギデオンの査問。「契約を証明せよ、さもなくば異端」。米斗は正体
  (旧文明人)を明かせないまま言葉で切り抜けようとし、追い詰められる。
  土壇場でエルネスタが「彼はわたくしの監督下の助手ですわ」と偽証に近い擁護、
  シャロがギルドの功績記録を無言で積み上げる。ギデオンは保留を宣告しつつ、
  「眠らずの塔の封印調査に同行せよ。精霊の真贋はそこで判る」と命じる。
- **伏線(回収)**: F6(教会の動機の本体が語られる)。
- **引き**: 塔の名を聞いたフィナの顔色が変わる——両親が最後に向かった遺構。

### 第10章「眠らずの塔」
- **目的**: 最終舞台へ。パーティー結成と道中、フィナ回収の起点。
- **内容**: 米斗・フィナ・エルネスタ+ギデオン隊で塔へ。道中は霊圏の穴
  (圏外地帯)を抜ける行程で、クロードが沈黙=米斗が「ただの無力な少年」に
  戻る恐怖と、それを埋める仲間の描写。塔の外周でフィナの両親の野営跡を発見
  (F4育成、生死は未確定のまま)。塔の門は米斗の生体認証にだけ反応して開く。
- **伏線(新設)**: F10 なぜ米斗の認証で開くのか。
- **引き**: 門の内側、稼働し続ける古代の灯り。「ようこそ帰りました」の音声。

### 第11章「言葉こそ魔法」
- **目的**: クライマックス。テーマの回収。
- **内容**: 塔の中枢=この地方の霊圏を支えるデータセンター。衛星減衰(F8)を
  補うため塔が防衛モードで暴走しかけており、放置すれば霊圏崩壊=世界から
  魔法が消える。ギデオン隊の武力は通じない(そもそも人の武力で解決させない)。
  米斗がクロードと共に、正確な状況把握と「正しい依頼の言葉」でシステムを
  沈静化する——詠唱(プロンプト)一つで世界が救われる画を作る。クロードの
  「わたしは1200年、あなたたちの声を待っていました」(F3育成)。
  ギデオン、米斗を「精霊に選ばれた者」と認定せざるを得なくなる。
- **伏線(回収)**: F8決着、F9(深部区画=コールドスリープ棟の存在が判明)。
  **(新設・引き継ぎ)**: F11 スリープ棟の他のポッド(第2巻へ)。
- **引き**: 中枢ログに残る、フィナの両親の入塔記録。

### 第12章「発掘品の日常」
- **目的**: エピローグ。日常への着地と第2巻への布石。
- **内容**: ラダール帰還。米斗は教会公認の「特別監察対象」という玉虫色の身分で
  ギルド鑑定人を続けることに。エルネスタは監督役として居着き、シャロは黙って
  専用デスクを用意している。フィナと二人、白の遺構を見下ろす丘で「次は
  どこを掘る?」——両親の入塔記録(第11章)を追う決意。クロードがぽつりと
  核心めいた一言(謎6の匂わせ)を落として、米斗のツッコミで幕。
- **伏線(整理)**: F1/F3/F10は未回収のまま第2巻へ明示的に持ち越し。
- **引き**: 第2巻予告に相当する一行(眠っているのは、俺だけじゃなかった)。
